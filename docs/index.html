A coding agent
you can put a leash on.

minder is a multi-provider (Anthropic, OpenAI, Gemini, Ollama) coding-agent harness written in Rust. Policy hooks are written in mq's embeddable query language rather than a general-purpose scripting language — it has no builtin for file writes, network requests, or process execution, so hooks can observe, block, or transform agent behavior without being able to cause side effects themselves.

What is minder?

The agent loop is a standard ReAct-style tool-calling loop; hooks only answer narrow policy questions at five fixed interception points and never drive the loop itself. Every tool call and result streams live to the terminal, and the same mq engine powers minder's autonomous loop mode — no external mq binary required.

Hooks that can't hurt you.

Most agent harnesses let you hook into the loop with a general-purpose scripting language — which means a hook can do anything the agent can do: write files, call the network, spawn processes. minder's hooks are written in mq, a jq-like query language with no builtin for any of that. A hook can only look at what's happening and answer a narrow policy question — it structurally cannot cause a side effect.

# .agent/hooks/circuit_breaker.mq

# stop the turn once 3 tool calls in a row have failed.

def on_context(ctx):

  let recent_failures = ctx.tool_results | last(3) | filter(is_error) | len()

  if(recent_failures >= 3, block("3 tool calls failed in a row"), allow())

block

No side effects by construction

mq-lang has no builtin for file writes, network requests, or process execution — not a sandboxed subset, an actual absence in the language.

Install

Requires a recent stable Rust toolchain if building from source.

cargo install minder-cli

minder "..."

git clone https://github.com/harehare/minder.git

cd minder

cargo install --path crates/minder-cli

minder "..."
